# QUIZAMO Mosaic Tool v1.5.0 - 配布版

## リリース概要
モザイクツールの完成版。カスタムアイコン付きのプロフェッショナルなElectronアプリケーション。

## 新機能・改善点

### 🎨 プロフェッショナルアイコン実装
- カスタムQロゴアイコンを実装
- 黒背景に白いQ輪郭、中央の穴部分にモザイクパターン
- 複数サイズ対応（16px〜1024px）
- モザイクツールの機能を視覚的に表現

### ⚡ 機能完成
- フォルダ選択によるローカル画像一括読み込み
- リアルタイムモザイク描画（3種類のエフェクト）
- 出力フォルダへの直接保存
- アンドゥ・リドゥ機能
- プロフェッショナルなUI/UX

### 🔧 技術仕様
- Electron 27.0.0 + Node.js
- HTML5 Canvas API
- セキュアなIPC通信
- ローカルファイルシステム統合

## 使用方法
1. `npm install` で依存関係をインストール
2. `npm start` でアプリケーションを起動
3. SELECT FOLDERで画像フォルダを選択
4. 出力フォルダを指定
5. ブラシでモザイクを描画
6. SAVEボタンで保存

## システム要件
- Windows 10以降
- Node.js 16以降
- 4GB以上のRAM推奨

## ファイル構成
- `main.js` - Electronメインプロセス
- `preload.js` - セキュアなIPC通信
- `index.html` - UIとモザイクエンジン
- `build/icons/` - アイコンファイル群
- `package.json` - アプリケーション設定
