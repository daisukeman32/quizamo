# QUIZAMO Mosaic Tool v1.5.0 配布版

## 📦 配布ファイル

### 実行ファイル
- **場所**: `dist/win-unpacked/QUIZAMO Mosaic Tool.exe`
- **アイコン**: カスタムQロゴ（黒背景、白Qアウトライン、モザイク中央穴）
- **バージョン**: 1.5.0
- **サイズ**: 約168MB

### 必要なファイル
`dist/win-unpacked/`フォルダ全体を配布してください。
EXEファイルだけでなく、同じフォルダ内のすべてのDLLとリソースファイルが必要です。

## 🚀 使用方法

1. **起動**: `QUIZAMO Mosaic Tool.exe`をダブルクリック
2. **フォルダ選択**: SELECT FOLDERボタンで画像フォルダを選択
3. **モザイク編集**: 
   - ブラシサイズ調整
   - モザイク強度設定
   - 3種類のモザイクタイプから選択
4. **保存**: SAVEボタンで完成フォルダに保存

## 🎨 機能

- **モザイクタイプ**:
  - Pixelate: ピクセル化モザイク
  - Blur: ぼかしモザイク
  - Mosaic: タイルモザイク
- **リアルタイムプレビュー**: カーソル位置にモザイク効果を表示
- **アンドゥ/リドゥ**: 最大50ステップの操作履歴
- **画像ナビゲーション**: 前後の画像に素早く移動

## 💻 システム要件

- Windows 10/11 (64-bit)
- メモリ: 4GB以上推奨
- ディスク容量: 200MB以上

## 🔧 トラブルシューティング

### アイコンが表示されない場合
Windowsのアイコンキャッシュの問題です。以下を試してください：
1. F5キーでエクスプローラーを更新
2. ファイルを別のフォルダに移動
3. PCを再起動

### GPUエラーが表示される場合
正常動作に影響はありません。無視して大丈夫です。
