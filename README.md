# クイザモ - 直感的モザイクツール v1.0

## 概要
画像に簡単にモザイク処理を施せる直感的なWebベースのツールです。

## 機能
- 📁 フォルダ単位での画像一括読み込み
- 🖌️ リアルタイム筆ツールでモザイク描画
- 🎨 3種類のモザイクエフェクト（ピクセル、ぼかし、モザイクタイル）
- 🔧 筆サイズとモザイク粗さの調整
- ↶ 元に戻す/やり直し機能
- 💾 編集した画像の個別保存
- ⬅️➡️ 画像間のナビゲーション

## 使い方
1. `mosaic-tool.html`をブラウザで開く
2. 「画像フォルダを選択」ボタンでフォルダを選択
3. マウスやタッチでモザイクを描画
4. 保存ボタンで編集済み画像をダウンロード

## 技術仕様
- HTML5 Canvas API
- Vanilla JavaScript
- レスポンシブデザイン対応
